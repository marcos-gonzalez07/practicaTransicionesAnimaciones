Practica de transiciones y animaciones

Transiciones con CSS

Permiten cambiar suavemente una propiedad CSS de un estado a otro durante un período de tiempo.

  • transition-property: Define qué propiedad cambiar
  • transition-duration: Define la duración del cambio. (s o ms)
  • transition-timing-function: Controla la aceleración o desaceleración.
    • linear → Velocidad constante.
    • ease (por defecto) → Comienza lenta, acelera y luego desacelera.
    • ease-in → Comienza lenta y acelera.
    • ease-out → Comienza rápida y desacelera.
    • ease-in-out → Comienza y termina lenta
  • transition-delay: Retrasa el inicio de la transición.

Transición de color de fondo

La transición de color de fondo en CSS permite cambiar suavemente el color de fondo de un elemento cuando se activa un cambio de estado, como un :hover. Esto crea un efecto visual agradable y fluido que mejora la experiencia del usuario al interactuar con la página. Por ejemplo, al pasar el cursor sobre un botón, su color de fondo puede cambiar de azul a rojo, atrayendo la atención del usuario sin ser abrupto.

Transicion de tamaño de fuente

La transición de tamaño de fuente facilita el cambio gradual del tamaño del texto de un elemento. Esto es útil para destacar elementos interactivos o importantes al aumentar su tamaño cuando se pasa el cursor sobre ellos. Por ejemplo, un párrafo de texto puede aumentar su tamaño de fuente cuando se sitúa el cursor sobre él, destacando su relevancia o indicando que es interactivo.

Transicion de opacidad

La transición de opacidad permite ajustar gradualmente la transparencia de un elemento. Esto es útil para efectos como desvanecimientos de entrada y salida, o para resaltar elementos al aumentar su opacidad. Por ejemplo, una imagen o un cuadro de texto puede pasar de ser semi-transparente a completamente opaco al situar el cursor sobre él, atrayendo la atención del usuario de manera sutil y elegante.

Transición de Posición

La transición de posición permite mover suavemente un elemento de una ubicación a otra dentro de la página. Esto se utiliza a menudo para crear efectos de desplazamiento o para animar elementos en respuesta a interacciones del usuario.

Animaciones

Permiten que los elementos cambien de estilo en varios estados definidos mediante @keyframes.

Propiedades básicas de las animaciones:

  • animation-name: Especifica el nombre de la animación (definido en @keyframes).
  • animation-duration: Establece la duración de la animación.
  • animation-timing-function: Define la curva de velocidad de la animación.
  • animation-iteration-count: Indica cuántas veces se repetirá la animación.
  • animation-direction: Define si la animación debe invertirse en cada ciclo.
  • animation-fill-mode: Establece cómo se aplica el estilo a los elementos antes y después de la animación.

Animacion de movimiento

La animación de movimiento en CSS utiliza la propiedad transform junto con @keyframes para mover elementos a lo largo del eje X o Y, creando un efecto de desplazamiento. Este tipo de animación es ideal para captar la atención del usuario y hacer que la interfaz sea más dinámica. Por ejemplo, un cuadro puede moverse horizontalmente de un lado a otro de la pantalla, lo que puede usarse para destacar información o guiar la mirada del usuario hacia un elemento importante.

Animación de Rotación

La animación de rotación permite que un elemento gire alrededor de su centro, utilizando la propiedad transform: rotate() en combinación con @keyframes. Este efecto es especialmente útil para añadir dinamismo a elementos como iconos, botones o gráficos, sugiriendo acción o cambio. Por ejemplo, un ícono de carga puede rotar continuamente para indicar que un proceso está en curso, o un elemento gráfico puede girar como parte de una animación decorativa.
